# 📈 Stock Market Price Prediction System (Adani Enterprises)

This project is a **Machine Learning-based Stock Market Prediction System** that predicts the **next trading day's Opening Price, Closing Price, High Price, and Low Price** using historical stock market data of **Adani Enterprises (AdaniENT)**.

The system is trained on historical **time-series stock data** and uses **supervised learning algorithms** to forecast future price movements. The goal of this project is to demonstrate how Machine Learning can be applied to financial market prediction.

---

## 🚀 Live Application

🔗 **Streamlit App:**
https://adanient-nextdaysharepriceprediction.streamlit.app/

---

## 📊 Project Overview

Stock market prediction is a challenging task due to the volatile and non-linear nature of financial markets. This project leverages historical stock data and machine learning models to analyze patterns and predict future prices.

The model takes historical stock price data as input and predicts:

* 📈 Opening Price
* 📉 Closing Price
* ⬆ High Price
* ⬇ Low Price

for the **next trading day**.

---

## 🧠 Machine Learning Workflow

1. **Data Collection**

   * Historical stock price dataset of **Adani Enterprises (AdaniENT)**.

2. **Data Preprocessing**

   * Handling missing values
   * Feature selection
   * Data formatting for model training

3. **Model Training**

   * Supervised learning model trained on historical price data.

4. **Prediction**

   * Model predicts next day's stock prices.

5. **Deployment**

   * Interactive web application built using **Streamlit**.
